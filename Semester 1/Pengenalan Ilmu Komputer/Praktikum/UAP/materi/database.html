Pengertian Database

Pengertian database yang paling sederhana adalah kumpulan dari tabel. Satu tabel merepresentasikan suatu entitas tertentu. Suatu entitas terdiri atas beberapa atribut.

  • Field/Kolom
  • Record/Baris

Microsoft Access

Microsoft Access adalah sebuah program aplikasi basis data komputer rasional. Yang ditujukan untuk kalangan rumahan dan perusahaan kecil hingga menengah. Sedikit berbeda dari Ms.Access, My Structured Query Language (MySQL) merupakan program yang berjalan sebagai server untuk menyediakan multi-user agar dapat mengakses ke sejumlah database. Salah satunya adalah website yang tentu saja membutuhkan database server atau hosting untuk menampung berbagai macam informasi.

Membuat Tabel Pada Microsoft Access:

  1. Buka Ms. Access
  2. Save dahulu pekerjaan anda
  3. Pilih Create, Group Tables lalu klik Table Design Isi Field untuk tabel Mahasiswa :
    • Ubah field size sesuai kebutuhan
    • Ctrl+S beri nama tabel "Data_Mahasiswa"
  4. Klik Data_Mahasiswa pada All Access Objects
  5. Pilih tab Field, lalu klik View dan pilih ’Datasheet View’ agar dapat mengisi record pada field
  6. Isikan record pada field yang tersedia

Membuat Database Di Command Prompt: MySQL

Sedikit berbeda dari Ms.Access, MySQL merupakan program yang berjalan sebagai server untuk menyediakan multi-user agar dapat mengakses ke sejumlah database. Salah satunya adalah website yang tentu saja membutuhkan database server atau hosting untuk menampung berbagai macam informasi.

Bahasa Basis Data (Database Language):

  1. Mendefinisikan Basis Data (Data Definition Language)
  2. Memanipulasi Basis Data (Data Manipulation Language)
  3. Melakukan operasi-operasi tertentu pada basis data

XAMPP:

Perangkat yang dapat menggabungkan 3 aplikasi kedalam satu paket (Apache, MySQL, dan PHPMyAdmin). XAMPP bisa digunakan untuk pengembangan website berbasis PHP dan juga sebagai server untuk lokal dalam pembuatan database dengan MySQL.

  1. Buka Aplikasi XAMPP
  2. Aktifkan MySQL dan Apache dengan cara klik tombol ‘Start’
  3. Masuk ke command prompt dengan cara klik tombol Windows + R kemudian ketikkan cmd lalu enter
  4. Masuk ke dalam sistem MySQL dengan mengetikkan kode berikut
  5. Jika berhasil, maka akan ada tulisan Welcome to the MySQL Monitor

Membuat Database Di CMD:

  1. Membuat Database
    2. create database namadatabase;

    Contoh: membuat database "data_mahasiswa"

    create database data_mahasiswa;
  2. Melihat daftar database
    3. show databases;
  3. Menggunakan database yang sudah dibuat
    4. use namadatabase;

    Contoh: Menggunakan database "mahasiswa"

    use data_mahasiswa;
  4. Membuat Tabel "informasi"
    5. Create table informasi (NPM varchar (3) primary key not null, nama varchar (20) not null, kelas varchar (2) not null);
  5. Melihat daftar – daftar table
    6. Show tables;
  6. Melihat Struktur table
    7. Describe namatabel;

    Contoh: Melihat Struktur table "informasi"

    describe informasi;
  7. Mengisi record dari Tabel "informasi"
    8. insert into informasi values ("108", "Nikita Azzahra", "3D");
  8. Memeriksa Kembali tabel yang sudah dibuat
    9. select * from informasi;
